# Download Directory Extension

Chrome の既定ダウンロードディレクトリの配下に、ダウンロード元ホストと ON にした時刻のサブディレクトリを作って保存する Chrome 拡張です。

## できること

- 拡張の ON/OFF をトグルで切り替える
- ON の間、`${DL元のホスト}/${YYYY-MM-DD}/${HH-mm}/` を保存先にする
- OFF にするまで、そのサブディレクトリへダウンロードする
- OFF の間は Chrome の通常のダウンロード先に保存する

拡張は Chrome のダウンロード先そのものは変更せず、ダウンロードファイル名に相対パスを付けます。

## インストール

1. Chrome で `chrome://extensions` を開く
2. 右上の「デベロッパー モード」をオンにする
3. 「パッケージ化されていない拡張機能を読み込む」を押す
4. このディレクトリを選ぶ

## 使い方

1. ツールバーの拡張アイコンを開く
2. トグルを ON にする
3. `${DL元のホスト}/${YYYY-MM-DD}/${HH-mm}/` 配下にダウンロードされる
4. 通常保存に戻す場合はトグルを OFF にする

## 補足

- 日付と時刻はトグルを ON にした時点のローカル日時です
- ホストはダウンロード URL の hostname を使い、取れない場合だけ referrer の hostname を使います
- ホストが取れないダウンロードは `unknown-host` 配下に保存します
- Chrome のダウンロード API では `:` を含むパスが無効になるため、時刻区切りは `-` にしています
- ON のまま日付や時刻が変わっても、保存先は切り替わりません
- もう一度新しいサブディレクトリにしたい場合は、OFF にしてから再度 ON にしてください
