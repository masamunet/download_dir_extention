# Download Directory Extension

Chrome の既定ダウンロードディレクトリの配下に、ホスト名と日付のサブフォルダを自動で付ける Chrome 拡張です。

## できること

- 通常の Chrome のまま保存
- `${YYYY_MM_DD}/${HOST_NAME}` で保存
- `${HOST_NAME}/${YYYY_MM_DD}` で保存
- 既定モードが通常でも、特定ホストだけ `${HOST_NAME}/${YYYY_MM_DD}` に固定

拡張は Chrome のダウンロード先そのものは変更せず、相対パスだけを上書きします。

## インストール

1. Chrome で `chrome://extensions` を開く
2. 右上の「デベロッパー モード」をオンにする
3. 「パッケージ化されていない拡張機能を読み込む」を押す
4. このディレクトリを選ぶ

## 使い方

1. ツールバーの拡張アイコンを開く
2. 保存先モードから次のいずれかを選ぶ
   - `通常`
   - `YYYY_MM_DD / HOST_NAME`
   - `HOST_NAME / YYYY_MM_DD`
3. 特定サイトだけ固定したい場合は、そのサイトを開いた状態で `このホストでは HOST_NAME / YYYY_MM_DD` を押す
4. 登録済みホスト一覧から個別解除、または一括解除できる

## 補足

- `HOST_NAME` はまず元ページの `referrer` の hostname を使い、取れない場合だけ実際のダウンロード URL から補完します
- 日付はダウンロード開始時点のローカル日付です
- `chrome://` などホストが取れないページでは通常保存になります
